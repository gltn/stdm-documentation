Renaming Spatial Entity Layer

The image below shows the Layers Panel of QGIS. The STDM layer name shown in the image below is not very appealing. To set a friendly display name, ensure the layer is selected.

Clickon the spatial unit manager.

Provide a friendly name on the change display name dialog as shown in the image below then click OK to change the name.

Note: Changing layer names through the Spatial Unit Manager only applies to layer in STDM database.
