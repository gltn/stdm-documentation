Entity in STDM refers to a database table in which a specific data is stored. Examples of entity in STDM include person, party, parcel, etc.

Each entity has an Entity Browser that loads any existing data from the database. The entity browser has buttons for adding, editing or deleting records (see the image below). You can also make searches based on searchable columns specified in the Configuration Wizard. An entity browser looks like the image below.

Entity Browser Tool buttons

- Add Records (Not available for spatial entities - to add a spatial record, you have to digitize a feature or import a record)

- Edit Record

- Delete Record

- View Document

Managing Entities refers to the process of adding, querying, editing and deleting records. The next sub-topics elaborate how to add, edit and delete records.

The tool buttons at the top may vary depending on the entity.
