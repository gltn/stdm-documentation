Modifying STDM Folders

Options module enables you to modify folders used by STDM.

You can change the supporting document folder, template folder and the output folder.

To modify the paths, you just have to click on the folder icon on each path as shown below and select a folder.

Once you have modified the folders, click on the Apply button ().

This will result in the reloading of all STDM modules to pick the changes in STDM folders.
