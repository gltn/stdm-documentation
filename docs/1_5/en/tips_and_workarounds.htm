Tips and Workarounds

While using the Document Designer you might face issues that could prevent you from using Document Designer. These issues are because of the limited flexibility of QGIS Print Composer on which the Document Designer is based on. This limitation will be resolved in QGIS 3.0.

These issues can be resolved through tips and workarounds that are listed below.

1. Do not generate a document using a template that is open in Document Designer. This will block you from adding any more items into the template. To fix the issue, close and open the Document Designer and open the template.

2. The attribute table formatting will be lost when re-opening a saved template. This is a QGIS Print Composer known issue. Thus, it is recommended to add the formatting of the attribute table before saving the template for the last time to avoid re-formatting it.
