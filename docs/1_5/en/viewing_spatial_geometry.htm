Viewing Extents of Spatial Entity Features

Once you have imported a spatial record or digitized it, you can view each added geometry using the map canvas. The Spatial Entity Browser is further customized to highlight and zoom into a record's feature that you have selected.

Follow the steps below to view the geometry of a selected record.

1. Go to the STDM Toolbar and click on Entities menu as shown below.

Alternatively, you can click on the STDM Menu located under QGIS Menu and click on the Entities sub-menu as shown in the image below.

2. Open an Entity Browser of any entity containing a geometry column.

Spatial Entity Browser opens, positioned in the bottom left corner of QGIS unlike non-spatial entity browser as shown in the image below. This is to allow better visibility to the Map Canvas.

4. Select a row to view a feature attached to a record as shown below.

When you select a row, its attached geometry appears in the map canvas as shown in the above image.

When you click on the Close button, the active layer gets removed from the map canvas.
