Export Data from the STDM

1. Select Export Data command item from the menu.

2. An export STDM Data pops up allowing you to choose the destination file and format.

3. Select the destination format from the list.

4.Browse to save location of the file (Destination File)

4. Enter a name for the file to be created.

5. Click Save on the file save dialog

6. Click Next to proceed.

If no queries are required, clear the Query Builder check box.

 11. Click Finish.

A message box will prompt the response when the action is completed. Dismiss the dialog.

Browse to the location of the file to see the results. If you are not satisfied with the result, the process can be repeated with new options.

12. Click on the OK button to finish the process.
