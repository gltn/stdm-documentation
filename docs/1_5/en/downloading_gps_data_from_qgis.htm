Downloading GPS Data from QGIS

To download a GPS data from a GPS, follow the steps below.

1. Connect the GPS to your computer (i.e. using USB cable).

2. Ensure the device is visible in your computer.

3. Start QGIS normally.

4. Launch the GPS plug-in from QGIS, Vector > GPS > GPS Tools as shown in the image below.

Note: For above data, the coordinate system used was: WGS 84.
